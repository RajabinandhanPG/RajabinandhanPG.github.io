LLM Platform

Self-Hosted Inference for a Distributed Team

What Was Built

Built and deployed a compartmentalized local LLM platform for 10+ remote users, replacing cloud API usage with a shared internal inference system.

  • Dual AMD Radeon AI PRO R9700 GPUs with 64GB combined VRAM.
  • Qwen3.6-35B via llama.cpp Vulkan backend.
  • LiteLLM proxy with usage tracking, access control, and multi-user routing.
  • Open WebUI for direct human use and API-backed agent access.
  • Localized model serving with no external LLM API dependency.

Operational Impact

  • Supported a shared 1M-token context window and 10 parallel sessions.
  • Delivered 500+ tok/s aggregate throughput and up to 92 tok/s per user.
  • Ran roughly 3x higher TPS than frontier cloud model workflows for targeted tasks.
  • Localized an estimated $8K-$10K in cloud API credits.
  • Reduced turnaround time on targeted inference tasks by roughly 70%.

ISOFINITI v2

Module 2C Voice Engine and Graphite RAG

Voice Engine Build

Built ISOFINITI v2 Module 2C on LXC 101 with RTX 5090 as a local speech and voice service layer.

  • Whisper-Large-v3 with CUDA for speech-to-text.
  • Kokoro TTS for local text-to-speech.
  • FastAPI service on :8091 for STT and :8092 for TTS.
  • Streaming transcript endpoint and health probe endpoint.
  • Bot-detection heuristics and install automation with working code.

Graphite RAG

Built a localized retrieval layer so knowledge workflows stayed inside firm-controlled infrastructure.

  • Localized document retrieval instead of cloud-managed knowledge tooling.
  • Compartmentalized internal retrieval workflows for secure knowledge access.
  • Designed to support internal team usage without sending documents off-platform.
  • Positioned as part of a broader local-first AI operations stack.
Infrastructure

Dual-Site Operations, Client Delivery, and Backup Design

Dual-Site Footprint

  • Operations spanning Illinois, USA and Tamil Nadu, India.
  • Three Proxmox hosts across the estate.
  • 10 segmented VLANs connected through UniFi Site Magic and WireGuard.
  • 31 virtual machines, 4+ LXC containers, and multiple shared internal services.
  • Designed for cross-border operations, backup resilience, and secure service isolation.

Fonespot Client Environment

  • 21 VMs across dev, QA, staging, and production.
  • Dedicated database servers and Jenkins CI/CD.
  • Network-isolated environments for safer client delivery.
  • 12+ application stacks and multi-host deployment patterns.
  • Structured to let a lean technical team support full-lifecycle delivery.

Storage, Backup, and Recovery

  • 3-2-1 style backup thinking with Proxmox Backup Server and Unraid NAS.
  • Cross-continent deduplicated backup replication patterns.
  • Designed for rapid recovery, cross-site protection, and low operational overhead.

AI and Rendering Extensions

  • RTX 5090 AI inference environment for FLUX.2, Hunyuan3D 2.1, and TRELLIS2.
  • Distributed render farm using Flamenco 3.9 and Blender.
  • On-prem compute footprint positioned as the equivalent of $10K+ monthly cloud GPU usage.
Platform Design

ISOFINITI Recruitment Platform Architecture

Platform Capabilities

  • Multi-tenant architecture with RBAC and role-specific access control.
  • BIPA-compliant consent logging and audit trail design.
  • Candidate intake across Naukri, Indeed, and email workflows.
  • Deduplication using normalized identifiers and SHA-256 hashing.
  • Zero-hardcoded-secrets architecture through managed configuration patterns.

Delivery and Team Enablement

  • Built internal infrastructure patterns that let the team operate on secure, localized systems.
  • Partnered on team composition, resource planning, and infrastructure budgeting.
  • Reduced tooling dependence while improving delivery control and privacy posture.
  • Used local-first AI as the enabling layer instead of an external dependency stack.
