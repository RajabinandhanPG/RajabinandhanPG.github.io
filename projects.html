Project 01

Multi-GPU LLM Inference Cluster

Technical Summary

  • Dual AMD Radeon AI PRO R9700 setup with 64GB combined VRAM.
  • Qwen3.6-35B deployed through llama.cpp Vulkan.
  • Shared 1M-token context window with 10 parallel sessions.
  • 500+ tok/s aggregate throughput and up to 92 tok/s per user.
  • Designed as a localized replacement for recurring cloud inference spend.

Why It Matters

  • Localized $8K-$10K in cloud API usage.
  • Delivered roughly 3x higher TPS than frontier cloud models for targeted workloads.
  • Created a reusable internal AI capability for developers, operational users, and automation flows.
Project 02

Module 2C Voice Engine

Build Details

  • LXC 101 with RTX 5090 as the compute node.
  • Whisper-Large-v3 with CUDA for STT.
  • Kokoro TTS for local TTS.
  • FastAPI on :8091 for STT and :8092 for TTS.
  • Streaming transcript endpoint, health probe, and bot-detection heuristics.
  • Install script and full working code delivered as part of the module.

Role in the Larger Stack

  • Extends ISOFINITI's internal AI infrastructure beyond text inference.
  • Keeps speech processing local and controllable.
  • Fits into the broader local-first architecture alongside the LLM platform and RAG services.
Project 03

Graphite RAG

Purpose

Built a localized retrieval-augmented generation environment for secure internal knowledge access.

  • Kept document workflows and retrieval inside firm-controlled infrastructure.
  • Supported internal knowledge access without pushing documents into external hosted systems.
  • Aligned with the broader compartmentalized architecture used across AI and infrastructure work.

Adjacent Automation Work

  • Connected to a broader pattern of operational automation using Google APIs and Dialpad API.
  • Fits the same design principle as local inference: retain control, reduce vendor dependence, preserve data locality.

Project 05

Why It Matters
